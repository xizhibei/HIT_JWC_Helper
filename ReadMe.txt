哈工大教务处小助手v1.1
主要功能：
1.登录教务处网站，计算成绩（包括学分绩以及GPA，当然，你必须在校内网）
	1.1.图表展示各学期分数情况
	1.2.导入导出当前成绩数据，并提供加密
	1.3.登录时可选保存密码，提供简单加密
2.待开发。。。

	85-89   3.7   
	82-84   3.3

	75-77   2.7   
	72-74   2.3   
	68-71   2.0   
	64-67   1.5   
	60-63   1.0   
	<60     0
